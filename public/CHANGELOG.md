# 更新日志

## v0.1.5 (2026-03-11)

### 新功能
- **AI 助手**：全新 AI 助手功能，支持多话题对话管理、流式消息响应、Markdown 渲染
  - 支持联网搜索（DuckDuckGo），可折叠搜索面板展示搜索结果
  - 支持自定义 API 地址、模型、系统提示词配置
  - 消息自动滚动、加载动画效果
- **管理员模式隐藏**：支持隐藏管理员界面，无需退出登录即可切换
- **侧边栏固定**：Stars 分组导航侧边栏支持固定展开

### 优化
- **性能优化**：使用 `React.lazy` + `Suspense` 懒加载和代码分割，减少首屏加载体积
- **流式消息**：改进 AI 助手消息流式处理和本地状态管理，优化自动滚动和清理逻辑
- **侧边栏样式**：优化 Stars 分组侧边栏宽度、圆角、过渡动画效果
- **UI 细节**：更新 StarRepoCard 组件样式和阴影效果
- **更新日志**：日志面板支持完整 Markdown 渲染

### 修复
- 修复 GitHub Stars API 获取中的无限循环问题

## v0.1.4 (2026-03-07)

### 新功能
- **Stars 页分组导航侧边栏**：左侧固定分组导航，点击快速定位到对应分组，自动高亮当前可见分组
  - 桌面端可折叠，状态持久化
  - 移动端浮动按钮 + 抽屉式导航
- **Stars 搜索关键词高亮**：搜索时在卡片上高亮匹配的仓库名、简介、Topics
- **Stars 搜索支持备注**：搜索范围新增用户填写的备注字段

### 优化
- **图标缓存优化**：记住成功获取图标的渠道索引，刷新后直接使用缓存渠道，不再重复请求失败的渠道
- **图标加载性能**：修复首次渲染时序问题，从 `useEffect` 异步读缓存改为 `useState` 同步初始化，刷新后 404 请求从 ~70 降为 0
- **全部失败缓存**：所有图标渠道都失败时缓存失败标记，下次直接显示字母图标，7 天后自动重试
